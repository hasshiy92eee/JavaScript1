☆シューティングゲーム操作方法☆

  • ゲームスタートボタンをクリックで1秒後に開始。
  • spaceキーで自機の弾発射です。
  • 矢印キーで移動します。「 ← ↑ ↓ → 」
  • ゲームオーバー時はRキーで再スタートです。
  • ゲームプレイ中は(Windows)ctrl+R
    (mac)command+Rでリセットです。
  • sキーでbgmのON/OFF
  • ゲーム終了はブラウザーの×です。
  • PCの性能によりゲームに
    遅延が発生することをご了承ください。
  • ※ゲームスタートボタンを連続でクリックするとゲームが高速化するので注意
